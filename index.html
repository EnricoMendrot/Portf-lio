FireYellow — IA de Visão Computacional para Detecção de Fogo e Fumaça

FireYellow é um sistema de Inteligência Artificial baseado em Visão Computacional, desenvolvido para detectar automaticamente focos de fogo e fumaça em imagens e vídeos. O projeto utiliza técnicas avançadas de Machine Learning e Deep Learning para atuação em cenários críticos, como áreas industriais e ambientais.

O modelo foi treinado com o dataset D-Fire e utiliza autoencoders para aprendizado de características, redução de ruído e aumento da robustez do sistema em diferentes condições ambientais.

ProjetoQueima — Sistema Inteligente de Monitoramento Corporativo

ProjetoQueima é uma solução desenvolvida para a empresa Petróleo Brasileiro S.A. (Petrobras), com foco no monitoramento, análise e visualização de dados operacionais. A aplicação foi construída para apoiar a tomada de decisão estratégica em ambientes corporativos de grande escala.

O sistema utiliza Django como framework principal no backend, Python para processamento de dados e bibliotecas especializadas em visualização gráfica, garantindo clareza, desempenho e confiabilidade.
